Automated Road Damage Detection Using UAV Images

Final Year Project

Developed a deep learning-based system that automatically detects and classifies road damage from Unmanned Aerial Vehicle (UAV) images. Infrastructure maintenance is expensive and manual inspection is slow — this system brings AI-powered automation to road monitoring at scale.

What it does

  • Detects road damage (cracks, potholes, surface defects) from aerial drone imagery with high precision.
  • Implements and compares three state-of-the-art YOLO architectures: YOLOv4, YOLOv5, and YOLOv7 on deep CNN backbones.
  • Trained and evaluated on the RDD2022 dataset (China road data) combined with a Spanish road dataset for cross-domain generalization.
  • Provides real-time object detection and localization in UAV images, suitable for deployment on edge devices or cloud inference.

Technical Approach

  • Used transfer learning on pre-trained YOLOv4/v5/v7 backbones fine-tuned on road damage imagery.
  • Applied data augmentation (flipping, rotation, brightness variation) to improve model robustness.
  • Benchmarked models using mAP, Precision, Recall, and F1 score metrics.
  • OpenCV used for image preprocessing and post-processing bounding box visualization.

Loan Insurance System Web Application

Full Stack

A complete loan management and insurance tracking web application built using the Django framework in Python. The system allows users to apply for loans, track repayment schedules, and manage insurance coverage in one unified platform.

What it does

  • User authentication and role-based access control for applicants, agents, and administrators.
  • Loan application workflow with status tracking — from submission to approval or rejection.
  • Insurance policy management linked to active loans with premium calculation logic.
  • Admin dashboard with overview of all loans, users, and payment records.
  • Responsive UI built with Bootstrap for accessibility on all devices.

Technical Details

  • Built on Django's MVT architecture, with clean separation of models, views, and templates.
  • SQLite3 as the primary relational database with Django ORM for all data operations.
  • Form validation and CSRF protection implemented using Django's built-in security features.
  • JavaScript used for dynamic form interactions and real-time field validation.

Web App
